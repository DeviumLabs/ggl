# Skill: SEO page check (Next.js Pages)

## Use when

- Creating or editing pages that should rank (category/product/institutional pages).

## Steps

1. Confirm title, meta description, canonical.
2. Confirm robots meta (index/follow for public pages; noindex for utility/404 if needed).
3. Confirm structured data (JSON-LD) when applicable (Product pages).
4. Confirm internal linking (Navbar links and breadcrumbs if present).
5. Confirm no duplicate content via multiple URLs (redirects + canonical align).

## Output

- A minimal patch (files changed + exact edits).
- A short checklist of what was verified after the change.
